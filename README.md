# homebrew-tap

Homebrew tap for [kargnas](https://github.com/kargnas) apps.

```sh
brew install --cask kargnas/tap/cctrans
```

| Cask | What it is |
|---|---|
| [`cctrans`](Casks/cctrans.rb) | [CCTrans](https://github.com/kargnas/cctrans) — macOS menu-bar translator, double `Cmd+C` to translate |

Cask versions are bumped automatically by the `update-tap` job in each
[cctrans release build](https://github.com/kargnas/cctrans/blob/main/.github/workflows/build-release.yml).
